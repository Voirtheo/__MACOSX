## Build Setup

``` bash
# 项目依赖安装
npm install

# 项目运行及端口号 localhost:2333
npm run dev

# 打包生成的项目
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# unit测试
npm run unit

# e2e测试
npm run e2e

# 运行所有测试
npm test
```
