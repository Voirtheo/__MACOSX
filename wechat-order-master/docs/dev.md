## 管理后台开发

![img](http://pbsp6zdxi.bkt.clouddn.com/pcshow.gif)

- ### 安装依赖
    ```bash
    # npm install
    ```

- ### 启动
    ```bash
    # npm run dev
    ```

- ### 打包部署
    ```
        $ npm run build
    ```

关于跨域问题的解决：
- **cors** （Cross Origin Resource）
- 在 dev 开发模式下可以下使用**webpack 的 proxy
```config.js
    proxyTable: {
    '/list': {
        target: 'http://api.xxxxxxxx.com',
        changeOrigin: true,
        pathRewrite: {
        '^/list': '/list'
        }
    }
    }
```
